# Job Hunter & Learner

Job Hunter & Learner is an AI-powered productivity and learning assistant built with **Next.js and TypeScript**.
The project integrates AI utilities that help users generate notes, estimate food nutrition, and receive diet suggestions through API-based AI services.

The goal of the project is to experiment with **AI-powered tools that improve learning, productivity, and daily decision making**.

---

## Features

### AI Utilities

* **AI Notes Generator**
  Generate summarized notes or study material automatically.

* **Calorie Estimator**
  Estimate calories and nutritional values for food items.

* **Diet Suggestions**
  AI-powered recommendations for diet planning.

### Dashboard

* Interactive UI components
* Modular dashboard layout
* Organized component structure

### API Routes

Custom backend API endpoints implemented using **Next.js App Router**:

* `/api/generate-notes`
* `/api/estimate-calories`
* `/api/diet-suggestion`

These APIs process user input and return AI-generated responses.
